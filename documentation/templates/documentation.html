This site was built using the Django framework (v2.0.7). Each section has been implemented within separate apps, each detailed in this documentation. The code is available on github.

Apps

  • accounts

    (User account registration, sign in/out and profile)

  • comic_strip

    (Build Customised comic strips)

  • tickets

    (Raise bugs/features with upvotes/comments)

  • forum

    (Define and comment on topics for discussion)

  • productivity

    (Graphs summarising tickets raised over time)

  • cart

    (New features and feature upvote purchases)

  • checkout

    (Process payments using Stripe)
